# GUESS THE NUMBER
.
## Features:
- A fun time killer to try out with friends
- A Two player game
- Displays the winner with scores
- Essential hints are given in the game.

## Usage:
- Both player selects a number.
- Whoever guesses the number is least tries will be the winner.
